# Live Lab (Prototype v1.0)
**Live Lab** is a software tool designed for practicing musical instruments. It enables dynamic exploration of musical elements, relationships, and lessons through an intuitive graphical interface.  

The project is intended for **technical documentation and internal development**, and serves as a foundation for research and educational applications related to music theory and learning systems.
